Reading Materials
Course Overview: 
Introduction to Computer Security and Cryptography
  • Reflections on Trusting Trust, Ken Thompson
  • Measuring Pay-per-Install: The Commoditization of Malware Distribution, by J. Caballero, C. Grier, C. Kreibich, V. Paxson

 
Principles of Secure Design

Assignment 01
  • SetUID Demystified, Chen, Dean, and Wagner, 2002.
  • Operating Systems Security, T. Jaegeri, 2008. Chapter 4, Security in Ordinary Operating Systems.

 

Cryptography before World War II

Assignment 02
  • Handbook of Applied Cryptography, by Menezes, van Oorschot, and Vanstone (CRC Press, 1997). The book is available for free download in PDF or Postscript format, from Alfred Menezes's website.
  • Applied Cryptography (2nd. edition), by Schneier

 

Modern Cryptography

Assignment 03
  • Practical Cryptography, by Ferguson and Schneier
  • Cryptography: Theory and Practice (3rd. edition), by Stinson

 
Basic Web Security Model

Assignment 04
  • Securing Browser Frame Communication. Adam Barth, Collin Jackson, and John C. Mitchell
  • The Security Architecture of the Chromium Browser. Adam Barth, Collin Jackson, Charles Reis, and the Google Chrome Team
  • Exposing private information by timing web applications. A. Bortz, D. Boneh, and P. Nandy

 
Web Application Security

Assignment 05
  • Cross site scripting explained, Amit Klein
  • SQL Injection attacks, Chris Anley
  • Robust Defenses for Cross-Site Request Forgery. Adam Barth, Collin Jackson, and John C. Mitchell

 
Session Management

Assignment 06
  • Secure Session Management With Cookies for Web Applications. Chris Palmer
  • Origin Cookies: Session Integrity for Web Applications. by Bortz et al.

 

Cryptography Review

Lab 1
  • The BREACH attack: encryption and compression don't mix, by Gluck, Harris, and Prado
  • The case for prefetching and prevalidating TLS server certificates, by E. Stark, L.S. Huang, et al.

 

HTTPS: goals and pitfalls

Assignment 07

Lab 2
  • ForceHTTPS: protecting high-security web sites from network attacks, by A. Barth and C. Jackson
  • The case for short-lived certificates. by Topalovic et al.

 

Security issues in Internet protocols

Assignment 08
  • A look back at Security Problems in the TCP/IP Protocol Suite, S. Bellovin, ACSAC 2004.
  • BGP Security in Partial Deployment, Lychev, Goldberg, Schapira, 2013.
  • DNS cache poisoning, Steve Friedl

 

Network Security Protocols

Assignment 09
  • A Security Evaluation of DNSSEC with NSEC3, J. Bau and J.C. Mitchell
  • Distributed Firewalls, S. Bellovin
  • Bro: A System for Detecting Network Intruders in Real-Time, V. Paxon

 

Unwanted traffic: Denial of service attacks
  • Details of a recent large-scale DDoS event (2013)
  • Practical network support for IP Traceback, S. Savage, et al.
  • A DoS-Limiting Network Architecture, Yang, Wetherall, and Anderson
